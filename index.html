Test Upload File CGI

Descirption

This website will get two or more txt files and store them into json, then display the content of each files. By using Python CGI(Common Gateway Interface) Script to read all the file have uploaded, and write a new json file and html file. Then redirect to new html which has a javascript function open the json file and print the output.
